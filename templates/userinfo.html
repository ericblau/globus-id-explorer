OpenID Connect apps can use the UserInfo interface to get more information about individuals. Globus's UserInfo interface returns the following information about you. Explanations follow the data below. To learn how to correct this data, click here.

{{oidcinfo}}

The data returned by the UserInfo interface is the same as the ID Token with the following addition.

Applications using the Globus Auth API may also request individuals' linked identities. This information is provided through the UserInfo interface and it appears as an identity set. (When you logged in, {{pagetitle}} requested access to your linked identities.) In the identity_set, the preferred_username is replaced by username. The meaning is the same.

Some identity providers—especially academic institutions and schools—report anonymized usernames to prevent misuse of your username. In the preferred_username or username field, you might see a random string of numbers and letters instead of your actual username. The same string will be used consistently every time you login.
